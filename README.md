**<h1 style="color:#E03E2D;">Redbyte</h1>**

**Programmable robots inside Minecraft**

Redbyte introduces robots, a programmable robotic entity that can be controlled using a simple custom language.
Design automation and create your own robotic helpers directly inside your world.

<span style="color:#E03E2D;">This mod is still in its very beginning phase.</span>

[Go to CurseForge](https://www.curseforge.com/minecraft/mc-mods/redbyte) - - - [Go to Discord](https://discord.gg/Qszbf3bAg7)

**✨ Features**

**🤖 Robo – a robot that can interact with the world**

**🧠 Simple Programming Language – control Robo with custom behavior scripts** _(documentations will be added soon)_

**🧪 Designed for experimentation, automation, and creative problem solving**

**🎮 Gameplay Focus**

Redbyte aims to become a tech progression & utility mod centered around robotics.

You can:

1. Prototype automation ideas
2. Create utility robots for tasks and world interaction
3. Build programmable systems instead of static machines

**🔥 Unique Concept**

Unlike traditional tech mods, Redbyte focuses on:

- Programmable behavior
- Player-driven logic instead of predefined automation
- Expandable tech ecosystem planned for future versions

**⚙️ Requirements**

Minecraft: 1.21.x

Loader: Forge

Multiplayer: Expected to work (report issues if found)
